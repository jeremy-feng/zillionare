## 大富翁量化框架

大富翁是可以本地部署的开源量化框架，自2019年起开发，2022年底2.0基本完成，功能齐全，能容纳超大规模数据（目前在生产环境已存储超30亿条行情数据）。

大富翁的起名有两重寓意，一是希望这个框架的使用者们，都能实现财富自由。另一方面，大富翁也是一款投资游戏的名字 -- 财富终究只是一场大富翁游戏。在这场游戏中，**努力和才能固然重要，但运气始终是不可或缺的**。
